# Pricing Game (GitHub Pages + Firebase)

## Quick start (you)

1) Copy the `docs/` folder into your GitHub repo (same structure).

2) In GitHub:
- Repo Settings → Pages
- Source: Deploy from branch
- Branch: `main`
- Folder: `/docs`

3) Wait 1–3 minutes, then open the GitHub Pages URL.

## Firebase prerequisites
- Firestore Database created
- Authentication enabled: Anonymous

## Using the app
- Instructor: open the Pages site and choose **Instructor** → **Create session**.
- Students: scan the QR code (shown on instructor lobby screen) to join.

## Notes
This is a minimal v1 implementation scaffold. You will tighten Firestore security rules after validating functionality.
